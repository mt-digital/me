Welcome!

Hello, welcome to my home page at the Northwest Knowledge Network at the University of Idaho. I am a research software developer interested in research data management and sharing, focusing on novel user interfaces and novel backend schemes for handling and generating metadata both about data and about models. I'm funded primarily by the WC-WAVE Project, with some extra funds from a USGS to work on a metadata editor.

See my GitHub repos to learn more about my work on WC-WAVE: vwplatform, wcwave_adaptors, and the adaptors documentation and associated repo.

To learn more about me, see the About page. See the "Tools and Tutorials" dropdown in the menu to see what tools I use and some information on how I use them.

News

June 24, 2015

I just returned from Indiana University where I attended a workshop for the Research Data Alliance Data Share Fellowship , which I was awarded at the end of May. My project with RDA will be to demonstrate the potential usefulness of linked open data in doing interdisciplinary research. For the current version of my project proposal, please see this version on github.

Before that, I traveled to Boise State University to participate in a two-week training course on interdisciplinary modeling in hydrology. We covered the economics of, social aspects, physical inputs, and parameters of hydrological models. As a final project, I put together a web app that allows a scientist to "run" a hydrological model through the web browser. Run is in quotes because in reality, the model has been pre-run for a set of predetermined inputs, and only those inputs may be selected by the user of the web app. See the github repository and the instructions there for installing and running the app.
